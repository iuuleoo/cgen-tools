Gato na Tela

Uma extensão leve para o Google Chrome

O que é

Esta extensão coloca um gatinho no seu navegador: ele aparece sobre as páginas que você visita, pode dormir, miar e ser arrastado para o canto da tela que você preferir. É só para deixar o dia um pouco mais simpático enquanto você trabalha ou estuda.

Código no GitHub

O projeto está aberto no GitHub: você pode ver o código, sugerir melhorias ou reportar problemas por lá. Substitua o link ab pelo endereço real do seu repositório.

Abrir repositório no GitHub

Como usar

  1. Baixe o pacote da extensão usando o botão abaixo (após aceitar os termos).
  2. No Chrome, abra chrome://extensions, ative o Modo do desenvolvedor.
  3. Clique em Carregar sem compactação e escolha a pasta da extensão descompactada.
  4. Fixe o ícone na barra de ferramentas, se quiser, e clique para ver o gato na tela.

Se você publicar na Chrome Web Store no futuro, pode trocar estas instruções por “instalar pela loja”.
